# team-player

curently in development!

React and Redux Example app.  

team-player is a fantasy football picker with a retro UX.


## Technologies

- React(React Dom, React Redux)
- Redux (Redux Logger,Redux Thunk)
- Sass (Node Sass)


## Features

- Comprehensive Redux State Management
- Mobile First
- 5 Player Filters
- Range sliders
- Async Data Calls
- Infinite scroll
- Flexbox
- BEM Naming
- 7-1 Sass
- Dynamic SVG
